# LangChain Learning

Learning agentic AI development by following the CampusX LangChain playlist.

## What's here
- **01-models** -- Groq, Ollama, Gemini integrations
- **02-prompts** -- PromptTemplate with save/load
- **03-chatbot** -- Multi-turn chatbot with message history
- **04-streamlit-app** -- Book summarizer web app
- **05-typedict-structured-review** -- Review Summary, settiments, pros and cons
- **06-pydantic-structured-review** -- Bettter than typedict

## Stack
Python, LangChain, Groq, Gemini, Ollama, Streamlit

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env  # add your API keys
```
